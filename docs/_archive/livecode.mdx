Example of how to implement the awesome live coding blocks in docusaurus

```jsx live
function Counter() {
  const [count, setCount] = React.useState(0);
  return (
    <div>
      <button onClick={() => setCount(count + 1)}>Click To Add</button>
      <h1>{count}</h1>
    </div>
  );
}
```
